# Weather Forecast App

Simple weather forecast single-page application built with HTML, Tailwind CSS, and JavaScript. Uses OpenWeatherMap for current weather and forecast data.

## Features
- Search weather by city name
- Use device geolocation for current location weather
- Recent searches dropdown (localStorage)
- Today's temperature toggle between 째C and 째F
- Custom alerts for extreme temperatures (>=40째C, <= -5째C)
- Dynamic background for rain/clouds/sunny
- 5-day forecast (aggregated from 3-hour forecast)
- Friendly UI messages and error handling

## Setup
1. Sign up at https://openweathermap.org/ and get a free API key.
2. Clone the repo:
   bash
